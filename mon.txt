این بخش مخصوی بیکاری در گروه😍🤣

لیست👇
با گفتن ( ply! ) لیست بازی ها برای شما ارسال میشه

با گفتن ( لینک ) ربات لینک گروه برای شما ارسال میکنه

باگفتن ( ادمین ) ربات لیست ادمین ها برای شما ارسال میکنه

باگفتن ( درباره گروه ) آنچه مدیر ها از ممبر خواستن ربات 
برای شما ارسال میکنه

(#بخش‌مدیر‌ها‌و‌ادمین‌هست )👇
مدیر و ادمین با گفتن ( آموزش ) اموزش کار با ربات یاد میگری

با گفتن (مشاهد متنی ) اموزش ساخت ربات میاد

باگفتن (مشاهد فیلمی ) اموزش ساخت ربات میاد

به زودی بخش های بیشتری به ربات اضافه میشه😉...
